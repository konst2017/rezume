Тестовое задание PHP разработчика

1. Функциональные требования
   1. Меню
      1. отображается на всех страницах;
      2. должен содержать элементы:
         1. Резюме - при клике должна открываться страница списка резюме;
         2. Мои резюме - при клике должна открываться страница моих резюме;
   2. Футер
      1. отображается на всех страницах;
   3. Страница списка резюме
Должна содержать следующие элементы:
      2. поисковая панель:
         1. поисковое поле;
         2. кнопка “Найти” - при клике должен происходить полнотекстовый поиск по резюме;
      3. панель вывода результатов поиска:
         1. количество найденных результатов;
         2. сортировка, возможные значения:
            1. По новизне - результаты должны сортироваться по убыванию даты изменения;
            2. По возрастанию зарплаты - результаты должны сортироваться по возрастанию цены;
            3. По убыванию зарплаты - результаты должны сортироваться по убыванию цены;
         3. элементы результата содержат:
            1. фото;
            2. дата обновления;
            3. специализация;
            4. зарплата;
            5. опыт работы;
            6. возраст;
            7. город проживания;
            8. последнее место работы;
               1. если последнего места работы нет, то отображать: “Без опыта работы”;
            9. при клике по элементу должна открываться страница просмотра резюме;
         4. пагинация;
      4. панель фильтрации:
         1. должна содержать следующие фильтры:
            1. выбор пола;
            2. город;
            3. зарплата;
            4. специализация;
            5. возраст от и до;
            6. опыт работы;
            7. тип занятости;
            8. график работы;
         2. при любом изменении фильтров должны обновляться результаты поиска;
         3. все параметры фильтрации и сортировки должны передаваться в get параметрах;
   4. Страница просмотра резюме
Должна содержать следующие элементы:
      5. фото
      6. специализация
      7. имя
      8. возраст
      9. занятость
      10. график работы
      11. город проживания
      12. контакты:
         1. электронная почта;
         2. телефон;
      13. информация об опыте работы:
         1. опыт работы;
         2. места работы:
            1. отсортирован по убыванию даты начала работы;
            2. каждый элемент должен состоять:
               1. дата начала и дата завершения:
                  1. если стоит галочка по настоящее время, то должно выводится: “по настоящее время”;
               2. продолжительность работы;
               3. организация;
               4. должность;
               5. Обязанности, функции, достижения;
      14. Обо мне;
   5. Страница моих резюме
Должна содержать следующие элементы:
      15. кнопка “Добавить резюме” - при клике должна открываться страница создания и редактирования резюме.
      16. список моих резюме:
         1. должны выводиться все существующие резюме в системе;
         2. каждый элемент должен содержать следующие элементы:
            1. специализация;
            2. зарплата;
            3. опыт работы;
            4. город работы;
            5. просмотры - количество открытий страницы резюме;
            6. дата публикации;
            7. кнопка “Открыть” - при клике должна открываться страница просмотра резюме;
            8. кнопка “Редактировать” - при клике должна открываться страница создания и редактирования резюме;
            9. кнопка “Удалить” - при клике должна удаляться вакансия;
   6. Страница создания и редактирования резюме
Должна содержать следующие элементы:
      17. Фото - обязательное;
      18. Фамилия - обязательное, текстовое;
      19. Имя - обязательное, текстовое;
      20. Отчество - обязательное, текстовое;
      21. Дата рождения - обязательное;
      22. Пол - выбор из значений:
         1. Мужской - выбран по умолчанию;
         2. Женский;
      23. Город проживания - обязательное, выпадающий список;
      24. Способы связи
         1. Электронная почта - обязательное, в формате электронной почты;
         2. Телефон - обязательное, в формате номера телефона;
      25. Желаемая должность
         1. Специализация - обязательное, выпадающий список, возможные значения приведены в перечислении 3.1. Специализации;
         2. Зарплата - обязательное, возможен ввод положительных целых чисел;
         3. Занятость - возможные значения:
            1. Полная занятость;
            2. Частичная занятость;
            3. Проектная/Временная работа;
            4. Волонтёрство;
            5. Стажировка;
         4. График работы - возможные значения:
            1. Полный день;
            2. Сменный график;
            3. Гибкий график;
            4. Удалённая работа;
            5. Вахтовый метод;
      26. Опыт работы
         1. опыт работы - выбор из значений:
            1. Нет опыта работы - при выборе должны скрываться места работы;
            2. Есть опыт работы - при выборе должны отображаться места работы;
         2. места работы:
            1. должны сортироваться по дате начала работы;
            2. элементы должны состоять из:
               1. Начало работы - должно состоять из:
                  1. месяц - обязательное, выпадающий список месяцев;
                  2. год - обязательное, возможен ввод значений от 1900 по текущий год;
               2. Окончания работы - должно состоять из:
                  1. месяц - обязательное, выпадающий список месяцев;
                  2. год - обязательное, возможен ввод значений от 1900 по текущий год;
                  3. По настоящее время:
                     1. не выбрано по умолчанию;
                     2. при выборе должны блокироваться для ввода и очищаться месяц и год окончания;
               3. Организация - обязательное текстовое поле;
               4. Должность - обязательное текстовое поле;
               5. Обязанности, функции, достижения - необязательное текстовое поле;
               6. кнопка “Удалить место работы” - при клике должно удаляться место работы;
            3. кнопка “Добавить место работы” - при клике должен добавляться элемент места работы;
      27. Расскажите о себе
         1. О себе - не обязательное;
      28. кнопка “Сохранить” - при клике должна происходит валидация данных:
         1. если есть ошибки, то они должны отобразиться;
         2. если ошибок нет, то резюме должно сохраниться и открыться страница моих резюме;
2. Не функциональные требования
   1. выполнять задание нужно на Yii 2 с использованием Yii 2.0 Basic Application Template;
   2. в качестве базы данных можно использовать MySql или PostgreSql;
   3. в работе нужно использоваться систему версионирования кода Git (gitlab.com ,github.com, bitbucket.org);
